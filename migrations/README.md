# Migrations Directory

This directory contains database migration scripts for the ZenSethu AI application.

## Migration Guidelines

1. Use a migration framework (e.g., Knex.js)
2. Number migrations sequentially
3. Include both up and down migrations
4. Document migration purpose and changes
5. Test migrations thoroughly
6. Backup data before running migrations

## Migration Categories

1. **Schema Migrations** - Database schema changes
2. **Data Migrations** - Data transformation scripts
3. **Seed Migrations** - Initial data population
4. **Rollback Migrations** - Reversal scripts
5. **Performance Migrations** - Index and optimization changes
6. **Security Migrations** - Security-related changes
