# Assets Directory

This directory contains all the static assets for the ZenSethu AI application.

## Directory Structure

- `icon.png` - App icon
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon
- `favicon.png` - Web favicon

## Guidelines

1. All images should be in PNG format
2. Icons should be at least 1024x1024 pixels
3. Splash screen should be 1242x2688 pixels for optimal display
4. Adaptive icons should follow Android guidelines
5. Favicon should be 48x48 pixels
