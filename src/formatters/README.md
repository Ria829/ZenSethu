# Formatters Directory

This directory contains application formatters for the ZenSethu AI application.

## Formatter Guidelines

1. Use consistent formatter patterns
2. Define clear formatter purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related formatters
6. Avoid circular dependencies

## Formatter Categories

1. **Data Formatters** - Data-related formatters
2. **State Formatters** - State-related formatters
3. **Cache Formatters** - Cache-related formatters
4. **Storage Formatters** - Storage-related formatters
5. **API Formatters** - API-related formatters
6. **Feature Formatters** - Feature-related formatters
