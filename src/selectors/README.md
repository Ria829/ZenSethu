# Selectors Directory

This directory contains data selection logic for the ZenSethu AI application.

## Selection Guidelines

1. Use efficient selection algorithms
2. Define clear input and output types
3. Include memoization where appropriate
4. Document selection logic
5. Test selectors thoroughly
6. Use consistent naming conventions

## Selection Categories

1. **State Selectors** - Application state selection
2. **Data Selectors** - Data filtering and sorting
3. **Computed Selectors** - Derived data calculations
4. **Aggregation Selectors** - Data aggregation logic
5. **Search Selectors** - Search and filtering logic
6. **Business Logic Selectors** - Business rule-based selection
