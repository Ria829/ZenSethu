# Coordinators Directory

This directory contains application coordinators for the ZenSethu AI application.

## Coordinator Guidelines

1. Use consistent coordinator patterns
2. Define clear coordinator purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related coordinators
6. Avoid circular dependencies
