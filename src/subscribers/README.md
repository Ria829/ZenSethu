# Subscribers Directory

This directory contains application subscribers for the ZenSethu AI application.

## Subscriber Guidelines

1. Use consistent subscriber patterns
2. Define clear subscriber purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related subscribers
6. Avoid circular dependencies

## Subscriber Categories

1. **Data Subscribers** - Data-related subscribers
2. **State Subscribers** - State-related subscribers
3. **Cache Subscribers** - Cache-related subscribers
4. **Storage Subscribers** - Storage-related subscribers
5. **API Subscribers** - API-related subscribers
6. **Feature Subscribers** - Feature-related subscribers
