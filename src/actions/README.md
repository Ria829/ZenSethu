# Actions Directory

This directory contains action creators for the ZenSethu AI application.

## Action Guidelines

1. Use consistent action naming
2. Define clear action payloads
3. Include action type definitions
4. Document action purposes
5. Test action creators
6. Use consistent naming conventions

## Action Categories

1. **State Actions** - Application state changes
2. **Data Actions** - Data manipulation actions
3. **UI Actions** - User interface actions
4. **Navigation Actions** - Navigation changes
5. **Form Actions** - Form-related actions
6. **Feature Actions** - Feature-specific actions
