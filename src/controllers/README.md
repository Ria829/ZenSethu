# Controllers Directory

This directory contains application controllers for the ZenSethu AI application.

## Controller Guidelines

1. Use consistent controller patterns
2. Define clear controller purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related controllers
6. Avoid circular dependencies

## Controller Categories

1. **Data Controllers** - Data-related controllers
2. **State Controllers** - State-related controllers
3. **Cache Controllers** - Cache-related controllers
4. **Storage Controllers** - Storage-related controllers
5. **API Controllers** - API-related controllers
6. **Feature Controllers** - Feature-related controllers
