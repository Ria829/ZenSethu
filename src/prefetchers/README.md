# Prefetchers Directory

This directory contains data prefetching logic for the ZenSethu AI application.

## Prefetcher Guidelines

1. Use consistent prefetching patterns
2. Define clear prefetching strategies
3. Include error handling
4. Document prefetching logic
5. Test prefetchers thoroughly
6. Use consistent naming conventions

## Prefetcher Categories

1. **Data Prefetchers** - Data prefetching
2. **API Prefetchers** - API response prefetching
3. **Cache Prefetchers** - Cached data prefetching
4. **Navigation Prefetchers** - Navigation data prefetching
5. **Feature Prefetchers** - Feature data prefetching
6. **User Prefetchers** - User-specific data prefetching
