# Profilers Directory

This directory contains application profilers for the ZenSethu AI application.

## Profiler Guidelines

1. Use consistent profiler patterns
2. Define clear profiler purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related profilers
6. Avoid circular dependencies

## Profiler Categories

1. **Data Profilers** - Data-related profilers
2. **State Profilers** - State-related profilers
3. **Cache Profilers** - Cache-related profilers
4. **Storage Profilers** - Storage-related profilers
5. **API Profilers** - API-related profilers
6. **Feature Profilers** - Feature-related profilers
