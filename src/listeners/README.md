# Listeners Directory

This directory contains application listeners for the ZenSethu AI application.

## Listener Guidelines

1. Use consistent listener patterns
2. Define clear listener purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related listeners
6. Avoid circular dependencies

## Listener Categories

1. **Data Listeners** - Data-related listeners
2. **State Listeners** - State-related listeners
3. **Cache Listeners** - Cache-related listeners
4. **Storage Listeners** - Storage-related listeners
5. **API Listeners** - API-related listeners
6. **Feature Listeners** - Feature-related listeners
