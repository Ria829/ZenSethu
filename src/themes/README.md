# Themes Directory

This directory contains theme configurations for the ZenSethu AI application.

## Theme Guidelines

1. Use a consistent color palette
2. Define typography scales
3. Include spacing systems
4. Support light and dark modes
5. Document theme variables
6. Use consistent naming conventions

## Theme Categories

1. **Color Themes** - Color palette definitions
2. **Typography Themes** - Font and text styles
3. **Spacing Themes** - Layout and spacing scales
4. **Component Themes** - Component-specific styles
5. **Dark Mode Themes** - Dark mode variations
6. **Accessibility Themes** - High contrast variations
