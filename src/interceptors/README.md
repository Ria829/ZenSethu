# Interceptors Directory

This directory contains HTTP interceptor logic for the ZenSethu AI application.

## Interceptor Guidelines

1. Use consistent interceptor patterns
2. Define clear interception points
3. Include error handling
4. Document interceptor logic
5. Test interceptors thoroughly
6. Use consistent naming conventions

## Interceptor Categories

1. **Authentication Interceptors** - Authentication handling
2. **Logging Interceptors** - Request/response logging
3. **Error Interceptors** - Error handling
4. **Caching Interceptors** - Response caching
5. **Retry Interceptors** - Request retry logic
6. **Performance Interceptors** - Performance monitoring
