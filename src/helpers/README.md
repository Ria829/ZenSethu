# Helpers Directory

This directory contains application helpers for the ZenSethu AI application.

## Helper Guidelines

1. Use consistent helper patterns
2. Define clear helper purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related helpers
6. Avoid circular dependencies

## Helper Categories

1. **Data Helpers** - Data-related helpers
2. **State Helpers** - State-related helpers
3. **Cache Helpers** - Cache-related helpers
4. **Storage Helpers** - Storage-related helpers
5. **API Helpers** - API-related helpers
6. **Feature Helpers** - Feature-related helpers
