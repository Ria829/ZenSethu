# Reducers Directory

This directory contains state reducer logic for the ZenSethu AI application.

## Reducer Guidelines

1. Use pure reducer functions
2. Define clear action types
3. Include comprehensive state updates
4. Document reducer logic
5. Test reducers thoroughly
6. Use consistent naming conventions

## Reducer Categories

1. **State Reducers** - Application state management
2. **Data Reducers** - Data manipulation logic
3. **UI Reducers** - User interface state
4. **Navigation Reducers** - Navigation state
5. **Form Reducers** - Form state management
6. **Feature Reducers** - Feature-specific state
