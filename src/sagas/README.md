# Sagas Directory

This directory contains side effect management logic for the ZenSethu AI application.

## Saga Guidelines

1. Use consistent saga patterns
2. Define clear side effect flows
3. Include error handling
4. Document saga logic
5. Test sagas thoroughly
6. Use consistent naming conventions

## Saga Categories

1. **API Sagas** - API call management
2. **Data Sagas** - Data synchronization
3. **Navigation Sagas** - Navigation side effects
4. **Form Sagas** - Form submission handling
5. **Notification Sagas** - Notification management
6. **Feature Sagas** - Feature-specific side effects
