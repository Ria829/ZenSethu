# Repositories Directory

This directory contains application repositories for the ZenSethu AI application.

## Repository Guidelines

1. Use consistent repository patterns
2. Define clear repository purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related repositories
6. Avoid circular dependencies

## Repository Categories

1. **Data Repositories** - Data-related repositories
2. **State Repositories** - State-related repositories
3. **Cache Repositories** - Cache-related repositories
4. **Storage Repositories** - Storage-related repositories
5. **API Repositories** - API-related repositories
6. **Feature Repositories** - Feature-related repositories
