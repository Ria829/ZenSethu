# Testers Directory

This directory contains application testers for the ZenSethu AI application.

## Tester Guidelines

1. Use consistent tester patterns
2. Define clear tester purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related testers
6. Avoid circular dependencies

## Tester Categories

1. **Data Testers** - Data-related testers
2. **State Testers** - State-related testers
3. **Cache Testers** - Cache-related testers
4. **Storage Testers** - Storage-related testers
5. **API Testers** - API-related testers
6. **Feature Testers** - Feature-related testers
