# Handlers Directory

This directory contains application handlers for the ZenSethu AI application.

## Handler Guidelines

1. Use consistent handler patterns
2. Define clear handler purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related handlers
6. Avoid circular dependencies

## Handler Categories

1. **Data Handlers** - Data-related handlers
2. **State Handlers** - State-related handlers
3. **Cache Handlers** - Cache-related handlers
4. **Storage Handlers** - Storage-related handlers
5. **API Handlers** - API-related handlers
6. **Feature Handlers** - Feature-related handlers
