# Models Directory

This directory contains application data models for the ZenSethu AI application.

## Model Guidelines

1. Use consistent model patterns
2. Define clear model purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related models
6. Avoid circular dependencies

## Model Categories

1. **Data Models** - Data-related models
2. **State Models** - State-related models
3. **Cache Models** - Cache-related models
4. **Storage Models** - Storage-related models
5. **API Models** - API-related models
6. **Feature Models** - Feature-related models
