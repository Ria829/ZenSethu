# Entities Directory

This directory contains application entities for the ZenSethu AI application.

## Entity Guidelines

1. Use consistent entity patterns
2. Define clear entity purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related entities
6. Avoid circular dependencies

## Entity Categories

1. **Data Entities** - Data-related entities
2. **State Entities** - State-related entities
3. **Cache Entities** - Cache-related entities
4. **Storage Entities** - Storage-related entities
5. **API Entities** - API-related entities
6. **Feature Entities** - Feature-related entities
