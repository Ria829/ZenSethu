# Dispatchers Directory

This directory contains application dispatchers for the ZenSethu AI application.

## Dispatcher Guidelines

1. Use consistent dispatcher patterns
2. Define clear dispatcher purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related dispatchers
6. Avoid circular dependencies

## Dispatcher Categories

1. **Data Dispatchers** - Data-related dispatchers
2. **State Dispatchers** - State-related dispatchers
3. **Cache Dispatchers** - Cache-related dispatchers
4. **Storage Dispatchers** - Storage-related dispatchers
5. **API Dispatchers** - API-related dispatchers
6. **Feature Dispatchers** - Feature-related dispatchers
