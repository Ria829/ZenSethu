# Routers Directory

This directory contains application routers for the ZenSethu AI application.

## Router Guidelines

1. Use consistent router patterns
2. Define clear router purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related routers
6. Avoid circular dependencies

## Router Categories

1. **Data Routers** - Data-related routers
2. **State Routers** - State-related routers
3. **Cache Routers** - Cache-related routers
4. **Storage Routers** - Storage-related routers
5. **API Routers** - API-related routers
6. **Feature Routers** - Feature-related routers
