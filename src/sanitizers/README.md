# Sanitizers Directory

This directory contains application sanitizers for the ZenSethu AI application.

## Sanitizer Guidelines

1. Use consistent sanitizer patterns
2. Define clear sanitizer purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related sanitizers
6. Avoid circular dependencies

## Sanitizer Categories

1. **Data Sanitizers** - Data-related sanitizers
2. **State Sanitizers** - State-related sanitizers
3. **Cache Sanitizers** - Cache-related sanitizers
4. **Storage Sanitizers** - Storage-related sanitizers
5. **API Sanitizers** - API-related sanitizers
6. **Feature Sanitizers** - Feature-related sanitizers
