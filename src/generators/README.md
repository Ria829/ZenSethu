# Generators Directory

This directory contains application generators for the ZenSethu AI application.

## Generator Guidelines

1. Use consistent generator patterns
2. Define clear generator purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related generators
6. Avoid circular dependencies

## Generator Categories

1. **Data Generators** - Data-related generators
2. **State Generators** - State-related generators
3. **Cache Generators** - Cache-related generators
4. **Storage Generators** - Storage-related generators
5. **API Generators** - API-related generators
6. **Feature Generators** - Feature-related generators
