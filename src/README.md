# Source Code Directory

This directory contains all the source code for the ZenSethu AI application.

## Directory Structure

- `components/` - Reusable UI components
- `screens/` - Screen components for navigation
- `services/` - Business logic and API services
- `contexts/` - React context providers
- `hooks/` - Custom React hooks
- `utils/` - Utility functions
- `assets/` - Static assets (images, fonts, etc.)
- `constants/` - Application constants
- `types/` - TypeScript type definitions

## Architecture

The application follows a modular architecture with clear separation of concerns:

1. **Screens** handle UI rendering and user interaction
2. **Services** encapsulate business logic and data operations
3. **Contexts** manage global state
4. **Components** provide reusable UI elements
5. **Hooks** encapsulate reusable logic
6. **Utils** provide helper functions
