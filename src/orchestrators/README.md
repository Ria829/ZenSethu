# Orchestrators Directory

This directory contains application orchestrators for the ZenSethu AI application.

## Orchestrator Guidelines

1. Use consistent orchestrator patterns
2. Define clear orchestrator purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related orchestrators
6. Avoid circular dependencies

## Orchestrator Categories

1. **Data Orchestrators** - Data-related orchestrators
2. **State Orchestrators** - State-related orchestrators
3. **Cache Orchestrators** - Cache-related orchestrators
4. **Storage Orchestrators** - Storage-related orchestrators
5. **API Orchestrators** - API-related orchestrators
6. **Feature Orchestrators** - Feature-related orchestrators
