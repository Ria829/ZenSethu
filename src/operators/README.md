# Operators Directory

This directory contains custom RxJS operators for the ZenSethu AI application.

## Operator Guidelines

1. Use consistent operator patterns
2. Define clear input and output types
3. Include error handling
4. Document operator logic
5. Test operators thoroughly
6. Use consistent naming conventions

## Operator Categories

1. **Data Operators** - Data transformation operators
2. **Validation Operators** - Validation logic operators
3. **Filtering Operators** - Data filtering operators
4. **Mapping Operators** - Data mapping operators
5. **Aggregation Operators** - Data aggregation operators
6. **Utility Operators** - General purpose operators
