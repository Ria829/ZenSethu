# Constants Directory

This directory contains application constants for the ZenSethu AI application.

## Constant Guidelines

1. Use consistent constant patterns
2. Define clear constant purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related constants
6. Avoid magic numbers

## Constant Categories

1. **Data Constants** - Data-related constants
2. **State Constants** - State-related constants
3. **Cache Constants** - Cache-related constants
4. **Storage Constants** - Storage-related constants
5. **API Constants** - API-related constants
6. **Feature Constants** - Feature-related constants
