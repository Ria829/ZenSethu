# Assets Directory

This directory contains static assets used throughout the ZenSethu AI application.

## Asset Guidelines

1. Use appropriate file formats (PNG for images, JSON for data)
2. Optimize assets for performance
3. Use consistent naming conventions
4. Organize assets in logical subdirectories
5. Include fallback assets for different screen sizes
6. Document asset usage and licensing

## Asset Categories

1. **Images** - Icons, illustrations, photos
2. **Icons** - Application icons and symbols
3. **Fonts** - Custom fonts and typography
4. **Data Files** - JSON data files
5. **Audio** - Sound effects and audio clips
6. **Video** - Video content (if applicable)
