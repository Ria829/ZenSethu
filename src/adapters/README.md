# Adapters Directory

This directory contains application adapters for the ZenSethu AI application.

## Adapter Guidelines

1. Use consistent adapter patterns
2. Define clear adapter purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related adapters
6. Avoid circular dependencies

## Adapter Categories

1. **Data Adapters** - Data-related adapters
2. **State Adapters** - State-related adapters
3. **Cache Adapters** - Cache-related adapters
4. **Storage Adapters** - Storage-related adapters
5. **API Adapters** - API-related adapters
6. **Feature Adapters** - Feature-related adapters
