# Monitors Directory

This directory contains application monitors for the ZenSethu AI application.

## Monitor Guidelines

1. Use consistent monitor patterns
2. Define clear monitor purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related monitors
6. Avoid circular dependencies

## Monitor Categories

1. **Data Monitors** - Data-related monitors
2. **State Monitors** - State-related monitors
3. **Cache Monitors** - Cache-related monitors
4. **Storage Monitors** - Storage-related monitors
5. **API Monitors** - API-related monitors
6. **Feature Monitors** - Feature-related monitors
