# Validators Directory

This directory contains application validators for the ZenSethu AI application.

## Validator Guidelines

1. Use consistent validator patterns
2. Define clear validator purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related validators
6. Avoid circular dependencies

## Validator Categories

1. **Data Validators** - Data-related validators
2. **State Validators** - State-related validators
3. **Cache Validators** - Cache-related validators
4. **Storage Validators** - Storage-related validators
5. **API Validators** - API-related validators
6. **Feature Validators** - Feature-related validators
