# Trackers Directory

This directory contains application trackers for the ZenSethu AI application.

## Tracker Guidelines

1. Use consistent tracker patterns
2. Define clear tracker purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related trackers
6. Avoid circular dependencies
