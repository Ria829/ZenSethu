# Epics Directory

This directory contains reactive programming logic for the ZenSethu AI application.

## Epic Guidelines

1. Use consistent epic patterns
2. Define clear stream transformations
3. Include error handling
4. Document epic logic
5. Test epics thoroughly
6. Use consistent naming conventions

## Epic Categories

1. **API Epics** - API call streams
2. **Data Epics** - Data flow management
3. **Navigation Epics** - Navigation stream handling
4. **Form Epics** - Form submission streams
5. **Notification Epics** - Notification streams
6. **Feature Epics** - Feature-specific streams
