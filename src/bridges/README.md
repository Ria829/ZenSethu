# Bridges Directory

This directory contains application bridges for the ZenSethu AI application.

## Bridge Guidelines

1. Use consistent bridge patterns
2. Define clear bridge purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related bridges
6. Avoid circular dependencies

## Bridge Categories

1. **Data Bridges** - Data-related bridges
2. **State Bridges** - State-related bridges
3. **Cache Bridges** - Cache-related bridges
4. **Storage Bridges** - Storage-related bridges
5. **API Bridges** - API-related bridges
6. **Feature Bridges** - Feature-related bridges
