# Publishers Directory

This directory contains application publishers for the ZenSethu AI application.

## Publisher Guidelines

1. Use consistent publisher patterns
2. Define clear publisher purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related publishers
6. Avoid circular dependencies

## Publisher Categories

1. **Data Publishers** - Data-related publishers
2. **State Publishers** - State-related publishers
3. **Cache Publishers** - Cache-related publishers
4. **Storage Publishers** - Storage-related publishers
5. **API Publishers** - API-related publishers
6. **Feature Publishers** - Feature-related publishers
