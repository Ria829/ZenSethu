# Proxies Directory

This directory contains application proxies for the ZenSethu AI application.

## Proxy Guidelines

1. Use consistent proxy patterns
2. Define clear proxy purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related proxies
6. Avoid circular dependencies
