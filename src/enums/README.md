# Enums Directory

This directory contains application enumerations for the ZenSethu AI application.

## Enum Guidelines

1. Use consistent enum patterns
2. Define clear enum purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related enums
6. Avoid duplicate values

## Enum Categories

1. **Data Enums** - Data-related enumerations
2. **State Enums** - State-related enumerations
3. **Cache Enums** - Cache-related enumerations
4. **Storage Enums** - Storage-related enumerations
5. **API Enums** - API-related enumerations
6. **Feature Enums** - Feature-related enumerations
