# Factories Directory

This directory contains application factories for the ZenSethu AI application.

## Factory Guidelines

1. Use consistent factory patterns
2. Define clear factory purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related factories
6. Avoid circular dependencies

## Factory Categories

1. **Data Factories** - Data-related factories
2. **State Factories** - State-related factories
3. **Cache Factories** - Cache-related factories
4. **Storage Factories** - Storage-related factories
5. **API Factories** - API-related factories
6. **Feature Factories** - Feature-related factories
