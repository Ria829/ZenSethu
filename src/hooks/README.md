# Hooks Directory

This directory contains custom React hooks used throughout the ZenSethu AI application.

## Hook Guidelines

1. Hooks should encapsulate reusable logic
2. Use TypeScript for type safety
3. Follow React hook rules
4. Export hooks with clear, descriptive names
5. Include TypeScript interfaces for parameters and return values
6. Write unit tests for complex hooks

## Common Hook Patterns

1. **Data Fetching Hooks** - Encapsulate API calls
2. **State Management Hooks** - Manage complex state logic
3. **Form Validation Hooks** - Handle form validation logic
4. **Device Interaction Hooks** - Manage device features (camera, location, etc.)
5. **Utility Hooks** - Encapsulate common utility functions
