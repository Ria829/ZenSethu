# Navigators Directory

This directory contains application navigators for the ZenSethu AI application.

## Navigator Guidelines

1. Use consistent navigator patterns
2. Define clear navigator purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related navigators
6. Avoid circular dependencies
