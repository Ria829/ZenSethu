# Loaders Directory

This directory contains data loading logic for the ZenSethu AI application.

## Loader Guidelines

1. Use consistent loader patterns
2. Define clear loading states
3. Include error handling
4. Document loader logic
5. Test loaders thoroughly
6. Use consistent naming conventions

## Loader Categories

1. **Data Loaders** - Data fetching and loading
2. **API Loaders** - API response loading
3. **Cache Loaders** - Cached data loading
4. **Fallback Loaders** - Fallback data loading
5. **Preloaders** - Data preloading
6. **Lazy Loaders** - Lazy data loading
