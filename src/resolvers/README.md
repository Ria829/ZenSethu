# Resolvers Directory

This directory contains data resolver logic for the ZenSethu AI application.

## Resolver Guidelines

1. Use consistent resolver patterns
2. Define clear data resolution logic
3. Include error handling
4. Document resolver logic
5. Test resolvers thoroughly
6. Use consistent naming conventions

## Resolver Categories

1. **Data Resolvers** - Data fetching and resolution
2. **API Resolvers** - API response resolution
3. **Cache Resolvers** - Cached data resolution
4. **Fallback Resolvers** - Fallback data resolution
5. **Transformation Resolvers** - Data transformation resolution
6. **Aggregation Resolvers** - Data aggregation resolution
