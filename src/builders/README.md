# Builders Directory

This directory contains application builders for the ZenSethu AI application.

## Builder Guidelines

1. Use consistent builder patterns
2. Define clear builder purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related builders
6. Avoid circular dependencies

## Builder Categories

1. **Data Builders** - Data-related builders
2. **State Builders** - State-related builders
3. **Cache Builders** - Cache-related builders
4. **Storage Builders** - Storage-related builders
5. **API Builders** - API-related builders
6. **Feature Builders** - Feature-related builders
