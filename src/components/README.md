# Components Directory

This directory contains reusable UI components used throughout the ZenSethu AI application.

## Component Guidelines

1. Components should be pure and reusable
2. Use TypeScript for type safety
3. Follow React best practices
4. Use consistent styling with the application theme
5. Export components with clear, descriptive names
6. Include PropTypes or TypeScript interfaces for props

## Component Categories

1. **UI Components** - Basic UI elements (buttons, inputs, cards)
2. **Layout Components** - Page layouts and containers
3. **Navigation Components** - Navigation-related UI
4. **Data Display Components** - Charts, lists, tables
5. **Feedback Components** - Alerts, toasts, modals
6. **Form Components** - Input forms and validation
