# Managers Directory

This directory contains application managers for the ZenSethu AI application.

## Manager Guidelines

1. Use consistent manager patterns
2. Define clear manager purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related managers
6. Avoid circular dependencies

## Manager Categories

1. **Data Managers** - Data-related managers
2. **State Managers** - State-related managers
3. **Cache Managers** - Cache-related managers
4. **Storage Managers** - Storage-related managers
5. **API Managers** - API-related managers
6. **Feature Managers** - Feature-related managers
