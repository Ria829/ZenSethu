# Services Directory

This directory contains application services for the ZenSethu AI application.

## Service Guidelines

1. Use consistent service patterns
2. Define clear service purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related services
6. Avoid circular dependencies

## Service Categories

1. **Data Services** - Data-related services
2. **State Services** - State-related services
3. **Cache Services** - Cache-related services
4. **Storage Services** - Storage-related services
5. **API Services** - API-related services
6. **Feature Services** - Feature-related services
