# Decorators Directory

This directory contains application decorators for the ZenSethu AI application.

## Decorator Guidelines

1. Use consistent decorator patterns
2. Define clear decorator purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related decorators
6. Avoid circular dependencies

## Decorator Categories

1. **Data Decorators** - Data-related decorators
2. **State Decorators** - State-related decorators
3. **Cache Decorators** - Cache-related decorators
4. **Storage Decorators** - Storage-related decorators
5. **API Decorators** - API-related decorators
6. **Feature Decorators** - Feature-related decorators
