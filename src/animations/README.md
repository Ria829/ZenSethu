# Animations Directory

This directory contains animation configurations for the ZenSethu AI application.

## Animation Guidelines

1. Use performant animation libraries
2. Define consistent easing functions
3. Include duration and delay settings
4. Document animation purposes
5. Test animations on different devices
6. Use consistent naming conventions

## Animation Categories

1. **Transition Animations** - Page and component transitions
2. **Loading Animations** - Progress and loading indicators
3. **Feedback Animations** - User interaction feedback
4. **Navigation Animations** - Navigation transitions
5. **Data Animations** - Data visualization animations
6. **Micro-Interactions** - Small UI animations
