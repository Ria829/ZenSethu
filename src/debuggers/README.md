# Debuggers Directory

This directory contains application debuggers for the ZenSethu AI application.

## Debugger Guidelines

1. Use consistent debugger patterns
2. Define clear debugger purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related debuggers
6. Avoid circular dependencies

## Debugger Categories

1. **Data Debuggers** - Data-related debuggers
2. **State Debuggers** - State-related debuggers
3. **Cache Debuggers** - Cache-related debuggers
4. **Storage Debuggers** - Storage-related debuggers
5. **API Debuggers** - API-related debuggers
6. **Feature Debuggers** - Feature-related debuggers
