# Middlewares Directory

This directory contains middleware logic for the ZenSethu AI application.

## Middleware Guidelines

1. Use consistent middleware patterns
2. Define clear middleware chains
3. Include error handling
4. Document middleware logic
5. Test middlewares thoroughly
6. Use consistent naming conventions

## Middleware Categories

1. **Authentication Middleware** - Authentication handling
2. **Logging Middleware** - Request/response logging
3. **Error Middleware** - Error handling
4. **Validation Middleware** - Request validation
5. **Caching Middleware** - Response caching
6. **Security Middleware** - Security checks
