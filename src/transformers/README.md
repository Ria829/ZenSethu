# Transformers Directory

This directory contains data transformation logic for the ZenSethu AI application.

## Transformation Guidelines

1. Use pure transformation functions
2. Define clear input and output types
3. Include error handling
4. Document transformation logic
5. Test transformations thoroughly
6. Use consistent naming conventions

## Transformation Categories

1. **API Transformers** - API response transformations
2. **Data Transformers** - Data structure transformations
3. **Format Transformers** - Data format conversions
4. **Normalization Transformers** - Data normalization
5. **Denormalization Transformers** - Data denormalization
6. **Business Logic Transformers** - Business rule transformations
