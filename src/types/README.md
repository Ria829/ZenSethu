# Types Directory

This directory contains application types for the ZenSethu AI application.

## Type Guidelines

1. Use consistent type patterns
2. Define clear type purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related types
6. Avoid circular dependencies

## Type Categories

1. **Data Types** - Data-related types
2. **State Types** - State-related types
3. **Cache Types** - Cache-related types
4. **Storage Types** - Storage-related types
5. **API Types** - API-related types
6. **Feature Types** - Feature-related types
