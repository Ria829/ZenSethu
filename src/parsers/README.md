# Parsers Directory

This directory contains application parsers for the ZenSethu AI application.

## Parser Guidelines

1. Use consistent parser patterns
2. Define clear parser purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related parsers
6. Avoid circular dependencies

## Parser Categories

1. **Data Parsers** - Data-related parsers
2. **State Parsers** - State-related parsers
3. **Cache Parsers** - Cache-related parsers
4. **Storage Parsers** - Storage-related parsers
5. **API Parsers** - API-related parsers
6. **Feature Parsers** - Feature-related parsers
