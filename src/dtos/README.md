# DTOs Directory

This directory contains application Data Transfer Objects for the ZenSethu AI application.

## DTO Guidelines

1. Use consistent DTO patterns
2. Define clear DTO purposes
3. Include documentation
4. Use consistent naming conventions
5. Group related DTOs
6. Avoid circular dependencies

## DTO Categories

1. **Data DTOs** - Data-related DTOs
2. **State DTOs** - State-related DTOs
3. **Cache DTOs** - Cache-related DTOs
4. **Storage DTOs** - Storage-related DTOs
5. **API DTOs** - API-related DTOs
6. **Feature DTOs** - Feature-related DTOs
