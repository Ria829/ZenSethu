# Guards Directory

This directory contains route guard logic for the ZenSethu AI application.

## Guard Guidelines

1. Use consistent guard patterns
2. Define clear guard conditions
3. Include error handling
4. Document guard logic
5. Test guards thoroughly
6. Use consistent naming conventions

## Guard Categories

1. **Authentication Guards** - Authentication checks
2. **Authorization Guards** - Authorization checks
3. **Navigation Guards** - Navigation restrictions
4. **Feature Guards** - Feature access controls
5. **Data Guards** - Data access restrictions
6. **Security Guards** - Security checks
