# Mocks Directory

This directory contains mock implementations for testing the ZenSethu AI application.

## Mock Guidelines

1. Use Jest's mock functionality
2. Create realistic mock data
3. Include both success and failure scenarios
4. Document mock behavior
5. Keep mocks up-to-date with real implementations
6. Use consistent naming conventions

## Mock Categories

1. **API Mocks** - Mock API responses
2. **Service Mocks** - Mock service implementations
3. **Component Mocks** - Mock component implementations
4. **Module Mocks** - Mock module exports
5. **Function Mocks** - Mock function implementations
6. **Data Mocks** - Mock data structures
