# Tests Directory

This directory contains unit and integration tests for the ZenSethu AI application.

## Testing Guidelines

1. Use Jest for unit testing
2. Use React Testing Library for component testing
3. Write tests for critical business logic
4. Mock external dependencies
5. Use descriptive test names
6. Include both positive and negative test cases

## Test Categories

1. **Unit Tests** - Test individual functions and components
2. **Integration Tests** - Test interactions between components
3. **Snapshot Tests** - Test UI consistency
4. **API Tests** - Test API integrations
5. **Navigation Tests** - Test navigation flows
6. **Service Tests** - Test service functionality
