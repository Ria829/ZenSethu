# Stories Directory

This directory contains Storybook stories for UI components in the ZenSethu AI application.

## Story Guidelines

1. Use Storybook for component documentation
2. Include all component variants
3. Use realistic data in stories
4. Document component props and usage
5. Include accessibility tests
6. Use consistent naming conventions

## Story Categories

1. **Component Stories** - Stories for individual components
2. **Page Stories** - Stories for complete pages
3. **Layout Stories** - Stories for layout components
4. **Form Stories** - Stories for form components
5. **Navigation Stories** - Stories for navigation components
6. **Utility Stories** - Stories for utility components
