# Locales Directory

This directory contains localization files for the ZenSethu AI application.

## Localization Guidelines

1. Use a consistent translation format (e.g., JSON)
2. Include all user-facing strings
3. Support multiple languages
4. Document translation keys
5. Validate translations for accuracy
6. Use consistent naming conventions

## Supported Languages

1. **English** - Default language
2. **Hindi** - Primary Indian language
3. **Tamil** - Southern Indian language
4. **Telugu** - Southern Indian language
5. **Bengali** - Eastern Indian language
6. **Marathi** - Western Indian language

## Translation Categories

1. **UI Strings** - User interface text
2. **Error Messages** - Error and validation messages
3. **Navigation Labels** - Navigation element text
4. **Form Labels** - Form field labels
5. **Button Text** - Button and action text
6. **Accessibility Text** - Accessibility descriptions
